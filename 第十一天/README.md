- 怎么定义rem和px关系
    - 假设设计图 1rem = 100px   （html的font-size:100px;）
    - 在375px宽度的设备上 那么需要换算一些这个html的font-size
    - 怎么换算呢？
 ```$xslt
    640      375
    ---   =  ----
    100      x=58.59375
 ```

- 写代码（比如说我一个元素宽是100px）
    640   1rem = 10px
    375   1rem = 58.59375px
